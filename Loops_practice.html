Types of Loops

Loop Type Description
for loop Loops a block of code a specific number of times
for...in loop Loops through the keys of an object
for...of loop Loops through the values of an iterable object (like an array)
while loop Loops while a specific condition is true. Note: The while loop does not support else if, but it supports if statements.
do...while loop Runs at least once, then continues while the condition is true
for loop (Example) for (let i = 0; i < 5; i++) {
  console.log("hello", i + 1);
}

For Loop Explanation

The syntax of a for loop looks like this: 

for (statement1; statement2; statement3) {
    // code to be executed
}
  • statement1 is executed once before the loop starts (usually initialization).
  • statement2 is the condition checked before each loop iteration.
  • statement3 is executed after each loop iteration (usually used for updating).
